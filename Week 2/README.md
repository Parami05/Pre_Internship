\# Week 2 - Sentiment Analysis using Machine Learning



\## Overview

In Week 2, a Machine Learning model was developed to analyze and classify text data based on sentiment.



\## Project Description

This project performs sentiment analysis on social media text data and categorizes it into:

\- Positive

\- Neutral

\- Negative



\## Workflow

\- Data cleaning

\- TF-IDF vectorization

\- Logistic Regression model



\## Results

\- Accuracy: \~91%



\## Technologies Used

\- Python

\- Pandas

\- Scikit-learn
