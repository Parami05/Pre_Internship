\#  Week 1 - Manga \& Manhua Library



\## Overview

As part of Week 1, a sample HTML project was created to design a visually appealing web interface.



\## Project Description

This project is a "Manga \& Manhua Library" web page that displays popular titles in a card-based layout with a modern dark theme.



\## Features

\-  Search bar for filtering content

\-  Category filters (All, Manga, Manhua)

\-  Image cards with titles and categories

\-  Clean dark-themed UI design



\## Technologies Used

\- HTML5

\- CSS3



\## Learning Outcome

\- Understanding of HTML structure

\- Practice in designing layouts using CSS

\- Improved UI design skills
